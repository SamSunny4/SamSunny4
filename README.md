## 🧩 Featured Projects

### 🔹 [KeyBase](https://github.com/SamSunny4/KeyBase)
> **Commercial Key Management System** *(Java, Swing, H2 Database)*  
A full-featured desktop application designed for key duplicator shops, actively deployed in a business firm. Manages customer records, handles embedded webcam captures, and provides sales analytics—all powered by a secure local H2 database and bundled as a standalone portable JRE application.

### 🔹 [Leadis / AiSam](https://github.com/SamSunny4/Leadis)
> **AI Developmental Screening Platform** *(Next.js, MediaPipe, Flask, ML)*  
An AI-powered web platform for early identification of learning disabilities in children. Integrates MediaPipe for in-browser gesture recognition and pose estimation, leveraging a Flask ML backend for multi-modal behavioral risk scoring. 

### 🔹 [DBMS](https://github.com/SamSunny4/DBMS)
> **Fraud Detection Platform** *(Next.js, Fastify, Neo4j, Three.js)*  
A full-stack blockchain monitoring tool utilizing a Neo4j graph database and a Fastify REST API. Features an interactive Graph Explorer with 2D and 3D visualizations for identifying transaction chains, computing risk profiles, and tracing multi-hop fraud patterns.

### 🔹 [FitMan](https://github.com/SamSunny4/FitMan)
> **Cross-Platform Gym Management System** *(C#, .NET MAUI, .NET 8.0)*  
An end-to-end multi-platform application (Windows, Android, iOS, macOS) with an MVVM architecture. Manages full member life cycles, versatile subscription tracking, check-ins, payment methods, and dashboard analytics.

### 🔹 [InfoGrid](https://github.com/SamSunny4/InfoGrid)
> **Smart Digital Display CMS** *(Next.js, TypeScript, MongoDB, Cloudflare R2)*  
A dynamic display board CMS optimized for vertical 43-inch indoor screens, featuring an auto-rotating feed of news, events, posters, and dynamic QR capabilities. Maintained securely via a robust admin panel.
